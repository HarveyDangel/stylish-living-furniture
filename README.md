# Stylish Living Furniture

A furniture brand website built with **Next.js 16**, **Tailwind CSS v4**, and the **Artisanal Hearth** design system.

## Design System

- **Typography:** EB Garamond (headings) paired with Hanken Grotesk (body/labels)
- **Colors:** Warm, earthy palette anchored in deep Umber (`#5D4037`), Ochre Sun (`#FBC02D`), and Ecru (`#F5F5DC`)
- **Layout:** Fixed 12-column grid (1280px max-width), 8px spacing scale, generous white space
- **Shapes:** Soft architectural — 4px default radius, 8px for cards

## Getting Started

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to see the result.

## Tech Stack

- **Framework:** Next.js 16 (Turbopack)
- **Styling:** Tailwind CSS v4 with `@theme inline` configuration
- **Fonts:** EB Garamond & Hanken Grotesk via `next/font/google`
